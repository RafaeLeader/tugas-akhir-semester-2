HTML (Hypertext Markup Language) adalah bahasa markah standar untuk membuat dan memformat halaman web. HTML digunakan untuk mengatur struktur dan konten suatu halaman web dengan menggunakan elemen-elemen yang ditandai dengan tag-tag. Bahasa markup standar yang digunakan untuk membuat halaman web dan aplikasi web.
