JavaScript



JavaScript adalah sebuah bahasa pemrograman tingkat tinggi yang digunakan untuk mengembangkan aplikasi web interaktif. Ia merupakan salah satu bahasa pemrograman paling populer di dunia dan banyak digunakan untuk mengubah tampilan dan perilaku halaman web. Berikut adalah penjelasan lengkap mengenai JavaScript:

  • Sintaksis: JavaScript menggunakan sintaks yang mirip dengan bahasa pemrograman C. Program JavaScript dapat ditulis di dalam tag script yang ditempatkan di dalam halaman HTML atau dapat disimpan dalam file eksternal dengan ekstensi .js dan dihubungkan ke halaman HTML
  • Tipe Data: JavaScript memiliki beberapa tipe data bawaan, termasuk Number (angka), String (teks), Boolean (benar/salah), Array (larik), Object (objek), dan lainnya. Tipe data dapat dikonversi satu sama lain jika diperlukan
  • Variabel: Variabel digunakan untuk menyimpan nilai dan diberikan nama yang unik. JavaScript menggunakan keyword var, let, atau const untuk mendeklarasikan variabel. var digunakan untuk mendeklarasikan variabel dengan cakupan fungsi, sedangkan let dan const digunakan untuk cakupan blok

contoh JavaScript:

  • document.writeln("Hello World")
  • let a = 5;
    let b = 10;
    let c = a + b; document.writeln(a, b, c)
  • let age = 18; if (age >=18) { console.log("Anda Sudah Dewasa"); } else { console.log("Anda Masih Di Bawah Umur") }


NOTE* console.log di lihat melalui inspect dengan mengklik kanan kursor lalu memilih pilihan console pada website
