CSS



CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen pada halaman web. Dengan menggunakan CSS, Anda dapat mengubah warna, ukuran, jenis font, tata letak, dan berbagai aspek visual lainnya pada elemen-elemen HTML, beberapa fungsi pada CSS:

  • Selektor: CSS menggunakan selektor untuk memilih elemen-elemen yang akan diberikan gaya. Selektor dapat berupa nama elemen, class, ID, atribut, atau kombinasi dari mereka. Contohnya, h1 untuk memilih semua elemen h1, .red-text untuk memilih elemen dengan class "red-text", #header untuk memilih elemen dengan ID "header".
  • Properti dan Nilai: Properti CSS digunakan untuk mengubah aspek visual dari elemen yang dipilih. Properti ditulis diikuti oleh titik dua (:), dan diikuti oleh nilai yang ingin diberikan. Contohnya, color: blue; untuk mengubah warna teks menjadi biru
  • Penerapan Gaya: CSS dapat di implementasikan dalam beberapa cara. Anda dapat menambahkan kode CSS di dalam tag style di dalam halaman HTML, atau menyimpannya dalam file eksternal dengan ekstensi .css dan menghubungkannya ke halaman HTML menggunakan tag


Beberapa contoh CSS:

  • h1 { color: blue; font-size: 35px; }
  • div class="text">

    Hello World

    style>
    .text { color: red;
    } (# bila menggunakan ID selain class)
    #green-text { color: green;
    }
    style>
  • body { background-color: #f2f2f2 }
